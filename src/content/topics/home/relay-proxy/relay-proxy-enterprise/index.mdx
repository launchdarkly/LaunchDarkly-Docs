## Overview

This topic explains what features are available for Enterprise users of the Relay Proxy.

<Callout intent="primary">
<CalloutTitle>Relay Proxy Enterprise is an Enterprise feature</CalloutTitle>
<CalloutDescription>

Relay Proxy Enterprise is available to customers on an Enterprise plan. To learn more, [read about our pricing](https://launchdarkly.com/pricing/). To upgrade your plan, [contact Sales](https://launchdarkly.com/contact-sales/).

</CalloutDescription>
</Callout>

## Understanding the differences between Relay Proxy and Relay Proxy Enterprise

The core set of features in the Relay Proxy is available to all customers, whether you're on a Starter, Professional, or Enterprise plan.

To learn more about what features are available to all customers, read [The Relay Proxy](/home/relay-proxy).

Enterprise customers can take advantage of additional features that make the Relay Proxy more secure and faster to configure.

We recommend that all customers on Enterprise plans use Relay Proxy Enterprise.

Here are the features available for Relay Proxy Enterprise:

* [Automatic configuration](/home/relay-proxy/automatic-configuration)
* [Offline mode](/home/relay-proxy/offline)
